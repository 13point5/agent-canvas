# Agent Canvas

A canvas application for coding agents. Bun monorepo with Hono server, React+Vite web app, and Commander CLI.

## Setup

```bash
bun install
```

## Run the server

```bash
bun run packages/server/src/index.ts
```

Server starts at http://localhost:3456. Test with:

```bash
curl http://localhost:3456/api/boards
```
